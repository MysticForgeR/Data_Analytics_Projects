# 🌫️ Delhi Air Quality Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** analyzing air quality data in Delhi.
The goal is to understand pollution trends, identify major contributors, and provide insights for better decision-making.

Air pollution is a critical issue in Delhi, and this dashboard helps visualize patterns in AQI (Air Quality Index) and pollutants over time.

## 🎯 Objectives

* Analyze air quality trends over time
* Identify the most harmful pollutants
* Compare pollution levels across locations
* Provide insights for environmental awareness

---

## 📂 Dataset

* Source: Public air quality dataset (Delhi AQI data)
* Data includes:

  * Date & Time
  * AQI values
  * Pollutants (PM2.5, PM10, NO2, CO, SO2, etc.)
  

---

## 📊 Dashboard Features

* 📈 AQI trend analysis over time
* 🧪 Pollutant distribution breakdown
* 📅 Time-based filtering (daily/monthly/yearly)
* 🔍 Interactive slicers for better analysis

---

## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning (Power Query)
* DAX (Data Analysis Expressions)

---

## 📌 Key Insights

* PM2.5 and PM10 are the major contributors to pollution
* Winter months show significantly higher AQI levels
* Certain locations consistently report poor air quality
* AQI trends show seasonal patterns
